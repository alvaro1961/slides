Black and White Images

Understanding Grayscale

  • Images are 2D arrays (matrices)
  • Each element = one pixel
  • Values represent intensity:
    • uint8: 0 (black) to 255 (white)
    • float: 0.0 to 1.0
  • Common to use uint8 when saving images
  • Float used during calculations

Color Images (RGB, Red Green Blue)

Understanding RGB

  • 3D arrays (height × width × 3)
  • Each pixel needs three numbers


\(2\times 2\) example

\[\begin{bmatrix}[255,0,0] & [0,255,0]\\ [0,255,0] & [100,100,100] \end{bmatrix}\]

  • Values per channel:
    • Red: controls red intensity (0-255)
    • Green: controls green intensity (0-255)
    • Blue: controls blue intensity (0-255)

Common Colors:

  • Black: (0, 0, 0)
  • White: (255, 255, 255)
  • Pure Red: (255, 0, 0)
  • Pure Green: (0, 255, 0)
  • Pure Blue: (0, 0, 255)
  • Yellow: (255, 255, 0) [Red + Green]
  • Magenta: (255, 0, 255) [Red + Blue]
  • Cyan: (0, 255, 255) [Green + Blue]

Accessing Color Channels in NumPy

Image Array Indexing:

  • Individual Pixels: image[ a, b, c ]
    • a, b: pixel coordinates (row, column)
    • c: color channel (0=red, 1=green, 2=blue)
    • Example: image[ 2, 3, 1 ] is the green intensity of the pixel in the third row and fourth column
  • Entire Color Channels: image[ : , : , c ]
    • Returns 2D array (0-255 intensities)
    • image[ : , : , 0 ] extracts red channel
    • image[ : , : , 1 ] extracts green channel
    • image[ : , : , 2 ] extracts blue channel

Reshaping for Matrix Operations

Why Reshape?

  1. Matrix operations need 2D arrays
  2. Color images are 3D
  3. Solution: Reorganize 3D → 2D
    • Original: height × width × 3
    • Reshaped: (height × width) × 3
  1. Use image.reshape(-1,3)
    • -1: automatically calculates rows (height × width)
    • 3: fixes 3 columns for RGB values
    • Each row becomes one pixel’s RGB values

Reshaping for Matrix Operations

Transformation Cycle

Image ⟶ Matrix

Start: Original Image - 3D array (height × width × 3) - Each pixel has RGB values


Convert to Matrix Form - Reshape to 2D: (height × width) × 3 - Each row is one pixel’s RGB values

Matrix ⟶ Image

Apply Matrix Operations - Linear transformations - Color adjustments - Filtering operations


Convert Back to Image - Reshape to 3D: height × width × 3 - Ready for visualization

Reshaping Back for Visualization

Back to Image Format

  1. After matrix operations, data is still 2D: \[\text{(height × width) × 3}\]
  2. Need 3D array for display: height × width × 3
  3. Solution: Reshape using original dimensions
  • image_modified.reshape(height, width, 3)
  • Must know original height and width!

Color Transformations

Switching Colors with Matrix Permutations

  • The image is an \(n\times 3\) matrix \(X\) (RGB values)
  • \(P\) is a \(3\times 3\) permutation matrix
  • \(X P\) rearanges the columns of \(X\)

Example: RGB ⟶ BGR

\[P = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 1 & 0 \\ 1 & 0 & 0 \end{pmatrix}\]

  • Multiplying by \(P\) swaps red and blue channels
  • Green channel stays in place

Color Transformations

RGB to BGR

RGB_to_BGR = np.array([

def swap_colors(image, permutation_matrix):
    # Reshape to 2D
    matrix = image.reshape(-1, 3)
    # Apply transformation
    transformed = matrix @ permutation_matrix
    # Reshape back to 3D
    return transformed.reshape(image.shape)

Grayscale Conversion

Human Perception

  • Eyes are not equally sensitive to all colors
  • Standard weights:
    • Red: 29.9%
    • Green: 58.7%
    • Blue: 11.4%
def to_grayscale(image):
    """Convert RGB image to grayscale"""
    weights = np.array([0.299, 0.587, 0.114])
    
    # Reshape and apply weights
    matrix = image.reshape(-1, 3)
    grayscale_values = matrix @ weights
    grayscale_values = np.clip(grayscale_values, 0, 1)  
    # Ensure valid range, since we use floats, range is [0,1]
    # Reshape to 2D (remove color channel)
    return grayscale_values.reshape(image.shape[:2])

Sepia Filter

Sepia Tone Transform

  • Transforms photos into warm brown tones, creating a vintage effect
  • Results in tones ranging from light cream to deep brown
def apply_sepia(image):
    sepia_matrix = np.array([
        [0.393, 0.349, 0.272],
        [0.769, 0.686, 0.534],
        [0.189, 0.168, 0.131]
    ])
    
    matrix = image.reshape(-1, 3)
    sepia = matrix @ sepia_matrix
    sepia = np.clip(sepia, 0, 1)  # Ensure valid range
    return sepia.reshape(image.shape)

Color Intensification

Enhancing Primary Colors

  • Intensifies RGB channels
  • Each channel increases its own weight
  • Reduces influence of other channels
def intensify_colors(image):
   intensity_matrix = np.array([
       [1.5, -0.2, -0.2],
       [-0.2, 1.5, -0.2],
       [-0.2, -0.2, 1.5]
   ])
   
   matrix = image.reshape(-1, 3)
   enhanced = matrix @ intensity_matrix
   enhanced = np.clip(enhanced, 0, 1)  # Ensure valid range
   return enhanced.reshape(image.shape)
